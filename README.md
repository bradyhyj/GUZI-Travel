# ✈️ GUZI.com (거지여행)
> **"설렘이 필요할 때, 내 방 침대 위에서 떠나는 전 세계 가상 여행"**

GUZI.com은 부산에서 출발하는 전 세계 주요 도시로의 여행을 가상으로 체험하고 계획할 수 있는 React 기반 웹 애플리케이션입니다. 실제 여행이 어려운 상황에서도 예산을 설정하고, 항공권과 숙소를 예약하며, AI가 제안하는 일정으로 여행의 기분을 만끽할 수 있는 '방구석 가상 여행' 서비스를 제공합니다.

---

## 🚀 주요 기능

### 1. 여행 검색 및 예산 관리
- **가상 도시 검색**: 부산(PUS)을 기점으로 도쿄, 뉴욕, 파리 등 11개 주요 도시 검색 가능.
- **실시간 예산 시스템**: 사용자가 설정한 총 예산에서 항공/숙소 비용을 차감하여 AI 플래너가 사용할 잔액을 실시간 계산.
- **데이터 영속성**: `localStorage`를 활용하여 새로고침 후에도 예약 내역과 예산 상태 유지.

### 2. 가상 예약 및 보딩 시스템
- **현실적인 예약 데이터**: 실제 항공사 정보와 '랜선 뷰맛집', '4K 유튜브 뷰' 등 위트 있는 가상 숙소 테마 결합.
- **Boarding Pass 발행**: 항공권과 숙소 예약이 일치하면 QR 코드가 포함된 가상 탑승권 UI 활성화.
- **비행 시뮬레이션**: 탑승 버튼 클릭 시 실제 목적지로 이동하는 듯한 비행기 이동 연출.

### 3. AI 일정 및 사후 기록
- **AI 일정 생성**: 도착한 도시의 예산에 맞춰 최적의 여행 경로 제안.
- **여행 리뷰**: 여행 종료 후 소감을 기록하거나 인스타그램 스타일의 레이아웃으로 여행 기록 공유.

## 🛠 기술 스택

### Frontend
- **Framework**: React 19 (Vite 기반)
- **State Management**: React Hooks (useState, useEffect)
- **Styling**: CSS-in-JS (Inline Styles) & App.css

### Libraries
- **Mapbox GL JS**: 지도 기반의 위치 정보 시각화
- **Turf.js**: 지리적 거리 및 위치 계산 로직
- **ESLint**: 코드 퀄리티 및 컨벤션 관리

---

## 📂 프로젝트 구조

- `src/App.jsx`: 애플리케이션 메인 라우팅 및 전역 상태 관리
- `src/Landing.jsx`: 메인 검색 창, 예산 설정 및 보딩패스 발행 UI
- `src/data.js`: 도시별 항공권 및 가상 숙소 모의 데이터 생성 로직
- `src/plan.jsx`: AI 기반 일정 생성 및 시뮬레이션 로직
- `src/insta.jsx`: 소셜 미디어 형태의 여행 기록 뷰
